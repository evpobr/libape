# ape_file_get_bytes_per_sample

Gets bytes per sample of APE file.

## Syntax

```c
APE_EXPORT int ape_file_get_bytes_per_sample(ape_file *ctx);
```

### Parameters

`ctx` **in**  
APE context.

## Return value

Return value >= `0` indicates success, error otherwise.
