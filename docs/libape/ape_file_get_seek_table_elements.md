# ape_file_get_seek_table_elements

Gets seek table elements count of APE file.

## Syntax

```c
APE_EXPORT int ape_file_get_seek_table_elements(ape_file *ctx);
```

### Parameters

`ctx` **in**  
APE context.

## Return value

Return value >= `0` indicates success, error otherwise.
