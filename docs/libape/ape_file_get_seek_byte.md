# ape_file_get_seek_byte

Gets seek byte of APE file.

## Syntax

```c
APE_EXPORT int ape_file_get_seek_table_elements(ape_file *ctx);
```

### Parameters

`ctx` **in**   
APE context.

`frame` **in**  
Frame to return seek byte.

## Return value

Return value >= `0` indicates success, error otherwise.
